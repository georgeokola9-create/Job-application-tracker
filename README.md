# Job Application Tracker

A personal job application tracker built as a portfolio project while practicing professional Git and GitHub workflows.

## Planned Stack

- Backend: Python, FastAPI, SQLAlchemy
- Database: SQLite for local development
- Frontend: HTML, CSS, JavaScript
- Workflow: feature branches, pull requests, and small conventional commits

## Security Notes

This project is designed to avoid committing sensitive information.

- Real environment variables go in `.env`, which is ignored by Git.
- Local SQLite database files are ignored by Git.
- `.env.example` is committed only as a safe template.
- Personal job application data should stay local unless intentionally anonymized.

## Planned Features

- Add, edit, delete, and list job applications
- Track company, role, status, dates, notes, and follow-up reminders
- Filter applications by status
- Dashboard summary of application progress
- Tests and GitHub Actions later in the project
